一、課程挑選

今天挑選課程時考量的點有：

  1. 是否能達成我的需求：使用 Jetpack Compose 開發、並能將舊 Java 專案轉換過來
  2. 字幕與口音是否易懂：使用倍速播放時能不受語言的限制
  3. 課程是否包含思考框架：希望不僅是學習工具，也從講師的經驗中學習開發方法
  4. 課程內容是否合時宜：雖想法與技術是從過去演化而來，但若會花大量額外時間，則應從現在所需為切入點，之後再視情況往前補，會更有效率。

因此挑了下面兩門課程：

  • The Complete Android 10 & Kotlin Development Masterclass: 以 Kotlin 為主，內容相對新，課程內容較多、有大型專案實做(Trello Clone)，整體學完應能掌握 Kotlin 基礎。

  • The Comprehensive 2020 Android Development Masterclass：以 Java 為主，課程應有超過兩年的歲數（當時最新版本為 Android 7.1），應適合學完現代開發方式後，用來補充 Java 與過去開發方式的課程

而除了上述兩門入門課外，亦有看了幾門未來可能學習的進階課程，都是由同一名講師Vasiliy Zukanov開的、皆獲得很高的評價，以下列做參考：

  • Android Architecture Masterclass
  • Android Unit Testing and Test Driven Development
  • Android Multithreading Masterclass
